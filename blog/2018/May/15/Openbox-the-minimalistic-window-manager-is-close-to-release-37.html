I have been using Openbox for quite a while and am now running the latest version compiled from a git checkout.

Had a little incident, but that was entirely my fault as I only did a openbox --restart which resulted in xscreensaver dying but still living in memory. I was unable to unlock the screensaver, so I had to log in remotely and reboot my computer.

If you are using Mageia Linux and want to be on the bleeding edge, you can install Cauldron and stay tuned for Openbox 3.7 when it lands.
